# Marist Parking Website
This repo contains my coursework for my System Design class. It contains a total of 3 separate works, but primarily focuses on 1: my front-end design for a hypothetical Marist Parking Website. 
<br> <br>
The files for the website are purely front-end (HTML, CSS, etc). The link to the website's homepage can be found on the right-hand side of this repo's main page.
The website is a hypothetical site that would track Marist University's parking lots and their congestion levels. Mock parking lot data were created and stored in JSON files 
to be fetched by the site's HTML scripts. They are then presented in such a way that the user can easily navigate the data for each
parking lot on campus to see how many spots in a lot are free, whether a spot is reserved or taken, etc.
<br> <br>
NOTE: <br>
- Not all of the links on the site lead to somewhere. Typically, only links associated with viewing parking lot data actually lead somewhere. Otherwise, you'll be redirected
to a 404 page.
- The files listed as filelist.html and fridge.html (and their associated CSS files and image files) are part of my other 2 (less important) works done for my System Design class.
